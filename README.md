😺 Mochi Web Agent

Record once. Schedule it. Get briefed on Telegram.

Mochi Web Agent is a no-code web automation tool for Windows. Record your browser actions—navigate, click, fill, collect—schedule them to run automatically, and receive AI-summarized results straight to your Telegram.

📥 Download & Installation (Releases)
- To get the latest version, please check the Releases section on the right side of this page.

- Download the .exe file from the latest release.

- No complex setup required—just run the executable to start automating.

- Check the release notes for the latest updates and patch info.

✨ Key Features
- No-Code Recording: Record clicks, inputs, and data collection in Chrome exactly as you perform them.

- Smart Scheduling: Set daily, weekly, or interval schedules to run your scenarios in the background.

- Telegram Briefing: Get AI-summarized results delivered to your Telegram automatically.

- Remote Control: Trigger any saved scenario on demand directly through Telegram messages.

💡 Use Cases
- News & Shopping: Daily digests of hot deals or keyword-based news monitoring.

- Market & Finance: Track prices, crypto balances, or exchange rates without manual checking.

- Work Automation: Monitor competitor updates or get daily business metric reports.
